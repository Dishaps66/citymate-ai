# Known Limitations

- Gemini/LangGraph orchestration is scaffolded but not connected.
- Supabase persistence handlers need project credentials and JWT verification middleware.
- Maps are Leaflet-ready, but route geometry rendering needs the map component pass.
- Public transport schedules require official GTFS or operator APIs per city.
- Accommodation requires real owner onboarding, moderation, and storage setup before public listings appear.
- Provider quotas and pricing should be verified before hackathon deployment.
