# Frontend Generation Prompt

Build a polished, evidence-first CityMate AI interface for a production Next.js app. The first screen must be the actual product dashboard, not a marketing-only landing page. Prioritize dense but readable operational UI, source metadata, timestamps, empty states, retry controls, maps, saved plans, verified listing badges, and accessibility controls. Never show fake city facts, fake properties, fake ratings, fake phone numbers, or fake schedules. When data is unavailable, show `Live verified data is currently unavailable for this location.`
